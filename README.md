# Mini ERP System

This Mini ERP System is designed for inventory management, facilitating the management of semi-finished goods, finished goods, packing materials, and product masters. It includes user management and system settings.

## Features
- User Management
- Product Master Management
- Data Entry Forms for Goods
- Reports
- CSS Styling

## Installation
1. Clone the repo.
2. Open the index.html file in a web browser.

## Usage
- Navigate through different modules.
- Enter data through forms.
- Generate reports as needed.
